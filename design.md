# RealtyInfo - UI Design Specification

## Design Philosophy
- **Clean & Minimal:** No clutter, only essential information
- **Instant Clarity:** Users should understand everything at a glance
- **Trust-Building:** Professional look inspires confidence in predictions
- **Mobile-First:** Works beautifully on all screen sizes

---

## Color Palette

```
Primary Colors:
- Brand Blue:     #2563EB (buttons, accents)
- Dark Text:      #1F2937 (headings, main text)
- Light Text:     #6B7280 (secondary text)
- Success Green:  #10B981 (positive growth percentages)

Background Colors:
- Page BG:        #F9FAFB (light gray)
- Card BG:        #FFFFFF (white)
- Input BG:       #FFFFFF (white)

Borders:
- Light Border:   #E5E7EB (card borders, dividers)
```

## Interactive Elements

### Hover States
```
Search Button:
- Default:  bg #2563EB
- Hover:    bg #1D4ED8 (darker blue)
- Active:   bg #1E40AF (even darker)

Property Cards:
- Default:  shadow-sm
- Hover:    shadow-md, transform translateY(-2px)
- Adds subtle lift effect

Autocomplete Items:
- Default:  bg white
- Hover:    bg #F9FAFB
```

### Focus States
```
Input Field:
- Default:  border #E5E7EB
- Focus:    border #2563EB, outline ring 2px #BFDBFE
```

### Animation Timing
```
- Card entrance:        fade-in 0.3s ease-out
- Hover lift:           transform 0.2s ease
- Number count-up:      1.0s ease-out (when results appear)
- Dropdown slide:       0.2s ease-out
```

## Design Principles Summary

1. **Clarity First**: Every element has a clear purpose
2. **Minimal Friction**: One input, instant results
3. **Visual Hierarchy**: Important info (prices) is largest
4. **Trust Signals**: Professional styling, clear disclaimers
5. **Responsive**: Beautiful on all devices
6. **Performant**: Fast load, smooth animations
